# 技能系統設計

技能採 **累積型 build + 隨機抽選 + 資料驅動**:技能 = 一筆資料,新增技能只需新增資料列,程式不用改。第一版實作 **12 種**,系統可無痛擴充到 ~50 種。

---

## 元素與狀態

**元素**:火 🔥 / 冰 ❄️ / 雷 ⚡ / 風 🌪️ / 土 ⛰️ / 毒 ☠️ / 物理 ⚔️(無屬性)

| 狀態 | 來源 | 效果 |
|---|---|---|
| 燃燒 | 🔥 | 持續傷害 |
| 緩速 / 凍結 | ❄️ | 移動 / 行動變慢,高層凍結 |
| 麻痺 | ⚡ | 短暫無法行動 |
| 擊退 | 🌪️ | 推開敵人 |
| 暈眩 | ⛰️ | 短暫無法行動 |
| 中毒 | ☠️ | 持續傷害,可疊層 |

> 元素與[裝備](equipment.md)的 `element` 連動:穿火屬裝備 → 火系技能加成,鼓勵玩家配 build。

## 技能資料結構

## 施放模型(自動施放)

主動技能**全自動施放**:冷卻一好就自動釋放並**自動瞄準**。玩家不手動按技能,操作重點放在**走位與閃避**(類《吸血鬼倖存者》)。

| 動作 | 控制 |
|---|---|
| 移動 | 手動(搖桿 / WASD) |
| **閃避(dash)** | 手動(一顆鈕,有無敵幀)—— 核心內建動作,不在抽選池 |
| 使用補血道具 | 手動(一顆鈕) |
| 所有主動技能 / 攻擊 | 自動(冷卻觸發 + 自動瞄準) |
| 被動技能 | 常駐 |

**自動瞄準**:投射物射最近敵人(`NEAREST`)、環繞技以自身為中心(`SELF`)、部分技朝移動方向(`MOVE_DIR`)。

---

數值用 `base + perLevel × 等級` 計算,升級只是等級 +1,效果自動成長。

```json
{
  "id": "fireball",
  "name": "火球",
  "category": "ACTIVE",        // ACTIVE 主動 / PASSIVE 被動 / BUFF 增益
  "element": "FIRE",
  "maxLevel": 5,
  "trigger": "AUTO_COOLDOWN",  // 冷卻好就自動施放(主動技能預設)
  "autoTarget": "NEAREST",     // 自動瞄準:NEAREST 最近敵人 / SELF 自身周圍 / MOVE_DIR 移動方向
  "cooldownMs": 3000,
  "effects": [
    { "type": "DAMAGE", "base": 20, "perLevel": 8 },
    { "type": "BURN",   "base": 5,  "perLevel": 2, "durationMs": 2000 }
  ]
}
```

## 第一版:12 種首發技能

### 主動 ACTIVE(綁按鍵 / 冷卻觸發)

| 技能 | 元素 | 效果 | 升級成長 |
|---|---|---|---|
| 火球 | 🔥 | 發射火球,傷害 + 燃燒 | 傷害↑、燃燒層數↑ |
| 冰錐 | ❄️ | 穿刺冰錐,傷害 + 緩速 | 傷害↑、緩速↑、可凍結 |
| 連鎖閃電 | ⚡ | 擊中後彈跳到附近敵人 | 彈跳數↑、傷害↑ |
| 風刃 | 🌪️ | 快速貫穿彈,擊退 | 貫穿數↑、攻速↑ |
| 落石 | ⛰️ | 範圍砸下,傷害 + 暈眩 | 範圍↑、暈眩時間↑ |
| 毒霧 | ☠️ | 放置毒區,範圍持續傷害 | 範圍↑、毒疊層↑ |
| 迴旋斬 | ⚔️ | 以自身為中心揮砍,範圍物理傷害(`SELF`) | 範圍↑、傷害↑ |

> **閃避(dash)** 為核心內建動作(手動、有無敵幀),不放進抽選池,確保每局都有可靠防禦。

### 被動 PASSIVE(常駐)

| 技能 | 元素 | 效果 | 升級成長 |
|---|---|---|---|
| 暴擊精通 | ⚔️ | 暴擊率 + 暴擊傷害提升 | 數值↑ |
| 嗜血 | ⚔️ | 命中敵人回少量血 | 吸血量↑ |
| 疾風步 | 🌪️ | 移動速度 + 攻擊速度提升 | 數值↑ |
| 烈焰護體 | 🔥 | 靠近的敵人持續燃燒 | 範圍↑、傷害↑ |
| 堅甲 | ⛰️ | 最大生命 + 防禦提升 | 數值↑ |

**範例 build**:純火爆發(火球 + 烈焰護體)、暴擊吸血肉搏(暴擊 + 嗜血 + 衝刺斬)、控場毒雷(毒霧 + 連鎖閃電)。

## 擴充到 ~50 的藍圖

每個「元素 × 機制」交叉 = 一個技能,共用同一套 `effects` 結構。

| 類別 | 擴充方向 |
|---|---|
| 主動·投射 | 散射火花、追蹤冰矢、雷球滯留、迴力風刃、毒鏢… |
| 主動·範圍 | 火焰新星、暴風雪、雷雨、地震波、酸蝕地面… |
| 主動·位移 | 閃現傳送、衝鋒撞擊、二段跳、瞬步反擊… |
| 被動·攻擊 | 連擊加成、處決、元素穿透、擊殺疊攻速… |
| 被動·防禦 | 受擊反傷、格擋、瀕死無敵、護盾自回… |
| 被動·協同 | 元素連鎖(燃燒中敵人受冰擊額外傷害…) |
| 增益 BUFF | 短時狂暴、護盾、時間緩慢、召喚分身 / 寵物 |

## 設計提醒

1. **不放「純大量回血」主動技能** —— 回血是[生活系統補血道具](life-system.md)的核心定位,避免架空。Sustain 一律條件式。
2. **元素協同是後期深度金礦** —— 第一版不做,但 `element` 欄位先留,日後加「火+冰=易碎、雷+水=感電」等連鎖,build 深度翻倍。
3. **技能特效策略** —— PixelLab 做投射物 / 技能卡圖等靜態小 sprite;命中爆炸、光環、拖尾、閃光、震屏用**程式粒子**做。一套基礎特效靠 element **換色重用**,50 技能不需 50 套 VFX。
