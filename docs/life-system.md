# 生活系統設計

戰鬥之外的**獨立家園畫面**:玩家在此採集、耕種、釣魚取得材料,合成**補血道具**與**裝備**、強化裝備。是裝備養成與補血的唯一來源。

---

## 三種生活活動

| 活動 | 玩法 | 產出 |
|---|---|---|
| **採集** | 在家園 / 地圖點擊資源點(樹、礦、草藥),有冷卻 | 木材、礦石、草藥… |
| **耕種** | 農場格子播種 → 等待成熟(`ready_at`)→ 收成 | 作物(蔬果、穀物…) |
| **釣魚** | 在水域釣魚(可做小互動 / 隨機) | 各種魚 |

> 第一版可從「簡化互動」起步(點擊 + 冷卻 / 計時),日後再加真實場景互動。

**時間模型:真實時間(real-time)** —— 作物成長、採集點復原依真實時間推進,離線也會成長,由伺服器時間判定。場景佈局、設施與擴建見 [hub.md](hub.md)。

## 合成系統

材料 → 成品,配方資料驅動(`recipe` 表),補血道具與裝備**共用**同一套合成機制。

```json
{
  "id": "recipe_potion_small",
  "output": { "itemId": "potion_small", "qty": 1 },
  "inputs": [
    { "itemId": "herb", "qty": 3 },
    { "itemId": "water", "qty": 1 }
  ]
}
```

### 合成產出類型

- **補血道具**:關卡內使用(見下)。
- **強化材料**:把裝備 `+1 → +10`(生活系統的主要養成產出)。
- **少數特定裝備**:次要;**裝備主要靠通關掉落取得**(見 [equipment.md](equipment.md) / [gameplay.md 裝備掉落](gameplay.md))。

---

## 補血道具

- **唯一回血來源**(技能不提供大量回血,維持生活系統的必要性)。
- 種類可分級:小 / 中 / 大補血,或附帶效果(回血 + 短時防禦)。
- 進關卡前由玩家攜帶(數量上限可設計),**戰鬥中手動使用**。

```json
{
  "id": "potion_small",
  "name": "小型補血藥水",
  "type": "HEAL",
  "healAmount": 30,
  "stackMax": 99
}
```

## 資源循環(整個遊戲的經濟核心)

```
採集 / 耕種 / 釣魚
        │  產出材料
        ▼
     合成 (recipe)
        ├─► 補血道具 ──► 關卡內使用(撐過 Boss 戰)
        └─► 強化材料 ──► 裝備 +1~+10

通關 Boss ──機率──► 裝備掉落 ──► 永久變強(主要取得方式)
```

- 戰鬥消耗補血道具 → 驅動玩家回家園生產。
- 想打更深的地圖 → 需要更強裝備 → **重刷舊地圖刷掉落** + 生活系統產**強化材料**養裝備。
- 形成「**戰鬥(刷裝)↔ 生活(養裝)**」的長期循環。
